# SeoulShelf — Korean skincare direct-from-Korea storefront

This is a polished static website prototype for a Korean skincare store that sells products sourced from Korea to U.S. customers with free shipping and slower delivery.

## What is included

- Responsive landing page
- Product catalog with filtering and search
- Cart drawer with localStorage persistence
- Direct-from-Korea value proposition
- DoorDash-style order status timeline
- Newsletter form prototype
- Modern K-beauty visual system

## How to open it

Open `index.html` in a browser.

No build tools are required.

## How to customize it

Edit `app.js` to change:
- Product names
- Prices
- Compare-at prices
- Categories
- Product descriptions
- Tracking statuses

Edit `styles.css` to change:
- Brand colors
- Typography
- Spacing
- Product card visuals

## Before using this for a real business

This is a front-end prototype, not a production ecommerce backend. Before launch, connect:

- Payment processor such as Stripe, Shopify Payments, or PayPal
- Product inventory and supplier data
- Order management system
- Customer emails and SMS tracking updates
- Taxes and sales tax calculation
- Returns/refunds workflow
- Import compliance, cosmetics labeling, customs, and ingredient documentation
- Privacy policy and terms of service

## Suggested next build step

Turn this static site into a Shopify theme, Webflow site, or Next.js ecommerce app with Stripe checkout and a real admin dashboard.
